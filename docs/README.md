# Stream the Spire - Documentation

User guides for setting up and running **Stream the Spire** - live card, relic, and potion toasts for **Slay the Spire 2** streams.

Download: [GitHub Releases](https://github.com/quality1441/sts2-stream-overlay/releases) (`stream-the-spire-win64-vX.Y.Z.zip`)

With the server running: **http://127.0.0.1:5055/help**

**Setup video:** [Watch on YouTube](setup-video.md)
